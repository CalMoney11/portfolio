My Work

Fullstack Ai Web app in progress

Art Buddy is a web app I built to give artists instant, constructive feedback without the wait. Using OpenAI's Vision models, it analyzes composition, color, and technique to deliver personalized critiques directly in the browser—replacing days of waiting for peer review with immediate, actionable insights. I designed the frontend with GitHub Pages and built a custom backend through Google Cloud Functions to handle secure image uploads, API calls, and automatic cleanup. What started as a simple ChatGPT embed has evolved into a full AI critique platform, and now I'm expanding it into a social hub where artists can post work, follow peers, and engage in AI-assisted discussions. The mission: help artists of all levels improve faster, together.

GIS and Organization
